在linux上配置固定ip地址

2012-12-12 14:15 3874人阅读 评论(0) 收藏 举报
分类：

版权声明：本文为博主原创文章，未经博主允许不得转载。

以redhat为例。

1.配置网卡IP地址

[java] view plaincopyprint?
  1. vi /etc/sysconfig/network-scripts/ifcfg-eth0  
vi /etc/sysconfig/network-scripts/ifcfg-eth0

内容如下：

[java] view plaincopyprint?
  1. DEVICE="eth0"  
  2. IPADDR=192.168.1.178         #固定IP地址  
  3. NETMASK=255.255.255.0        #子网掩码  
  4. GATEWAY=192.168.1.1          #网关  
  5. HWADDR="00:0C:29:0E:98:81"   #默认值，如果没有忽略  
  6. NM_CONTROLLED="yes"          #默认值，如果没有忽略  
  7. ONBOOT="yes"                 #必须为yes  
DEVICE="eth0"
IPADDR=192.168.1.178         #固定IP地址
NETMASK=255.255.255.0        #子网掩码
GATEWAY=192.168.1.1          #网关
HWADDR="00:0C:29:0E:98:81"   #默认值，如果没有忽略
NM_CONTROLLED="yes"          #默认值，如果没有忽略
ONBOOT="yes"                 #必须为yes
2.配置网卡DNS

[java] view plaincopyprint?
  1. vi /etc/resolv.conf  
vi /etc/resolv.conf

内容如下：

[java] view plaincopyprint?
  1. nameserver 211.161.159.5     #首选DNS  
  2. nameserver 202.103.24.68     #次DNS  
  3. search localdomain  
nameserver 211.161.159.5     #首选DNS
nameserver 202.103.24.68     #次DNS
search localdomain

然后运行

[java] view plaincopyprint?
  1. service network restart  
service network restart
